マップフォルダのディレクトリ構造について
-----------------------------------------
ワールドデータや YML ファイルを含むフォルダのディレクトリ構造は次のようになります::

	<マップ名>
	  │  config.yml
	  └─<マップ名>
	      │  level.dat
	      └─region

.. _siyou:

マップ読み込みの仕様について
-----------------------------
マップは、 ``plugins/KokuminTeamPvP/maps`` フォルダ内からプラグインの(マップではありません) config.yml に指定されている順番に読み込まれます。

config.yml (例)::

	map-rotation:
	  - 'Map 1'
	  - 'Map 2'
	  - 'Map 3'

また、読み込み時には ``plugins/KokuminTeamPvP/maps`` フォルダからマップフォルダ(:ref:`参照 <kouzou>`)がコピーされます。
あるマップに変更を加える場合、コピー元フォルダの方を変更しなければなりません。
変更を加えた場合は次にそのマップが読み込まれるタイミングで更新されます。
